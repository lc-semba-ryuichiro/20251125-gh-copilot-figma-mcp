# Positivus デザインガイドライン

Positivus デザインシステムの基本スタイルガイドです。Figma デザインに基づいて定義されています。

## 概要

このデザインシステムは以下の要素で構成されています：

- **タイポグラフィ**: Space Grotesk フォントファミリーを使用
- **カラーパレット**: 緑（アクセント）、ダーク、グレーの3色を基調
- **イラストレーション**: Igor Kapustin によるイラスト素材
