## CSS カスタムプロパティ

```css
:root {
  --color-green: #B9FF66;
  --color-dark: #191A23;
  --color-grey: #F3F3F3;
  --color-black: #000000;
  --color-white: #FFFFFF;
}
```

## 使用例

| カラー | 用途 |
|--------|------|
| Green | ラベル背景、ターシャリボタン、アクセント |
| Dark | プライマリボタン、テキスト、フッター背景 |
| Grey | セクション背景、カード背景 |
| Black | テキスト |
| White | 背景、ダーク背景上のテキスト |
