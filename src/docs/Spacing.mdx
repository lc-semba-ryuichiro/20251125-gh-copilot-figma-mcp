## CSS カスタムプロパティ

```css
:root {
  /* Spacing */
  --spacing-xs: 5px;
  --spacing-sm: 10px;
  --spacing-md: 20px;
  --spacing-lg: 40px;
  --spacing-xl: 60px;
  --spacing-2xl: 100px;

  /* Border Radius */
  --radius-sm: 7px;
  --radius-md: 14px;
  --radius-lg: 45px;

  /* Container */
  --container-max-width: 1240px;
  --container-padding: 100px;

  /* Shadows */
  --shadow-card: 0 5px 0 0 var(--color-dark);
}
```
