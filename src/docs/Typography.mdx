# タイポグラフィ

Positivus デザインシステムで使用するタイポグラフィです。フォントファミリーは **Space Grotesk** を使用しています。

## フォントファミリー

```css
font-family: "Space Grotesk", sans-serif;
```

[Google Fonts で Space Grotesk を見る](https://fonts.google.com/specimen/Space+Grotesk)

## CSS カスタムプロパティ

```css
:root {
  /* Typography - Font Family */
  --font-family: "Space Grotesk", sans-serif;

  /* Typography - Font Weights */
  --font-weight-regular: 400;
  --font-weight-medium: 500;

  /* Typography - Font Sizes (Desktop) */
  --font-size-h1: 60px;
  --font-size-h2: 40px;
  --font-size-h3: 30px;
  --font-size-h4: 20px;
  --font-size-body: 18px;

  /* Typography - Font Sizes (Mobile) */
  --font-size-h1-mobile: 43px;
  --font-size-h2-mobile: 36px;
  --font-size-h3-mobile: 26px;
  --font-size-h4-mobile: 18px;
  --font-size-body-mobile: 16px;
}
```
